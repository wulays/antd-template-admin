# 文档说明

> 项目技术：react typescript sass zustand antd swr 等

## 环境

-   nodejs version 18.12.0
-   pnpm 8.4.0

## 资源

[图标库](https://icon-sets.iconify.design/ant-design/)
[文档](https://ant.design/)

## 项目启动

依赖安装

```shell
pnpm install
```

项目运行

```shell
pnpm dev
```

项目构建

```shell
# 正式服
pnpm build
# 测试服
pnpm build:test
```
